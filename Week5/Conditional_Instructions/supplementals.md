Coming from higher level programming, loops are a little more confusing in assembly. I was worried I wasn't going to pick the correct processor flag or that my syntax was not correct. Having to compile the program everytime to debug it is a little bit of a pain. Though, the first task was simple enough.
On the other hand, task 2 was a lot more work. Trying to keep track of where everything is while coding is hard and you only know if you've done it right once you step through the code. Finding the correct conditional jump for task 2 was difficult.
